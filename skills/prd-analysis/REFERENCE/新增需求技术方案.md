## 6. 业务流程

### 6.1 核心流程

```mermaid
sequenceDiagram
    participant 用户
    participant 系统
    participant 数据库

    用户->>系统: POST /api/xxx
    系统->>系统: 参数校验
    系统->>数据库: 保存数据
    数据库-->>系统: 返回结果
    系统-->>用户: 返回创建的记录
```

### 6.2 异常处理

| 场景 | HTTP状态码 | 错误码 | 错误信息 |

| 参数校验失败 | 400 | INVALID_PARAM | 参数错误 |
| 数据不存在 | 404 | NOT_FOUND | 数据不存在 |
| 重复数据 | 409 | DUPLICATE | 数据已存在 |
